# 架构摘要

v0.1.0 只维护一份英文主技术文档，本文件提供中文摘要，避免早期同时维护两份大型技术文档。

当前工程分层：

- `cli.py`：Typer 命令、参数解析和面向用户的输出。
- `i18n/`：JSON 翻译资源与轻量查询函数。
- `device.py`：设备策略与本机能力检测。
- `doctor.py`：结构化环境检查，未来可直接扩展 JSON 输出。
- `model_registry.py`：模型槽位占位，不下载、不加载模型。
- `schemas.py`：未来任务、模型、对齐结果和导出结果的数据模型。

未来真实对齐能力应放在独立 `alignment/` 模块中，由 CLI 构造 job manifest 后调用，不应把模型推理逻辑写进 CLI。
