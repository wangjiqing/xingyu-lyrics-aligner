# Audio-to-Lyrics Transcription Spike

This spike has been promoted to formal optional scripts.

Use the repository-root entrypoints instead:

```bash
python scripts/extract_candidate_lyrics_from_audio.py --help
python scripts/normalize_candidate_lyrics_script.py --help
```

The old spike entrypoints remain as compatibility wrappers:

```bash
python spikes/audio_to_lyrics_transcription/run_spike.py --help
python spikes/audio_to_lyrics_transcription/normalize_transcript_script.py --help
```
